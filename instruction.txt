Закинуть содержимое этой папки в проект. 
В src рапределить файлы соответственно названию 
Прописать в коммандной строке npm i (установит все что лежит package.json
Прописать npm run (соотв команду) 
Готово

Команды:
1. dev - работа в режиме разработки(dev)
2. build - работа в режиме продакшена
3. zip - собирает проект в zip
4. deploy - закидывает на FTP сервер (в config указать данные)
5. svgSprive - собирает svg спрайты
